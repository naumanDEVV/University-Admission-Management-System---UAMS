# University Admission Management System (UAMS)

A simple **console-based University Admission Management System** developed in **C#** to practice **Object-Oriented Programming (OOP)** concepts.

This project simulates a basic university admission process including student registration, merit calculation, degree allocation, subject registration, and fee calculation.

---

## Features

- Add Degree Programs
- Add Students with program preferences
- Merit Calculation (FSC + ECAT marks)
- Automatic Admission Allocation
- Register Subjects for Students
- View Registered Students
- View Students in a Specific Degree Program
- Fee Calculation based on registered subjects

---

## Technologies Used

- **Language:** C#
- **Framework:** .NET Console Application
- **Concepts:** Object-Oriented Programming (OOP)

---

## OOP Concepts Implemented

This project demonstrates:

- Classes and Objects
- Encapsulation
- Lists (`List<T>`)
- Object Interaction
- Basic Sorting Algorithm (Bubble Sort)
